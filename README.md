### 🔨 Stuff I've Made

**[PairSpace](https://github.com/vbhati15/pairspace)** — a real-time collaborative code editor built on CRDTs (Y.js) instead of naive diffing, so multiple people can edit the same file without conflict resolution nightmares. Currently adding Docker-sandboxed code execution so you can actually *run* code inside a shared session, not just write it.
`React` `Vite` `Monaco Editor` `Y.js` `Docker`

**[Remote Browser Control](https://github.com/vbhati15/Remote_browser_control)** — a system for controlling a real browser session remotely over WebSockets using the Chrome DevTools Protocol.
`TypeScript` `Next.js` `Docker` `CDP`

**[Personal Finance Tracker](https://github.com/vbhati15/Personal_finance_tracker)** — a full-stack app for logging expenses, categorizing spend, and visualizing where money actually goes, deployed live on Vercel.
`MongoDB` `Express` `React` `Node.js`

**[Task Manager](https://github.com/vbhati15/Task_manager)** — a MERN task-management app built around the basics done right: clean CRUD, sensible state management, and an interface that doesn't get in the way of actually getting things done.
`MongoDB` `Express` `React` `Node.js`

**[Smart Resume Ranker](https://github.com/vbhati15/smart-resume-ranker)** — an ML/NLP tool that scores and ranks resumes against a job description, built to cut through the noise of manual screening.
`Python` `NLP` `Machine Learning`
